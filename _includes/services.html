How I get things done

Determining the problem

Identifying what needs to be done in what order. I like to start by getting a basic outline of what I want done

Plan of attack

I figure out how im going to get these things done, make a gameplan and follow it.

Getting it Done

Whether its coding, or building something, following the plan always helps to not mess things up.

Fit and Finish
