Blackjack

This was a project for a object oriented programming class which i chose to recreate blackjack.

I decided to code this in python for its ease of use and the fact that its capable of machine learning which I have a lot of interest in.

The idea was fairly simple, create an array for a card that contained all the suits and numbers in a deck.

Then I created the deck which includes a counter to ensure there is 52 cards, a function to shuffle the cards, as well as a function to return a card if it is a duplicate and delete it, and finally a function to deal the cards.

The final class was the game class. This contains all of the game rules, gives you the option to stay, hit, or quit, and much more. To simplify the game i decided to make face cards all worth 10 and an ace being either 1 or 11 depending on your hand count. This class also keeps track of the games previously played and shows counters for either wins or losses.
